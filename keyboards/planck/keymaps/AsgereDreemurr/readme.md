# The Default Planck Layout

## Overview
### Music mode
The following keys have a certain functionality:
- LCTL - start a recording
- LALT - stop recording/stop playing
- LGUI - play recording
- KC_UP - speed-up playback
- KC_DOWN - slow-down playback

## Development
Build firmware  
`qmk compile`  

Flash keyboard  
`qmk flash`
